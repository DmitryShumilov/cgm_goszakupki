## 🔄 Процесс обновления

Обновление данных происходит из Excel файла `database.xlsx` в базу данных PostgreSQL.

### Вариант 1: Быстрое обновление (рекомендуется)

**Шаг 1:** Обновите Excel файл
- Откройте `database.xlsx`
- Добавьте новые данные
- Сохраните файл

**Шаг 2:** Запустите скрипт обновления

```powershell
# Откройте PowerShell в папке проекта
cd C:\Dashboards\cgm_goszakupки

# Запустите скрипт
python import_excel_to_pg.py
```

**Шаг 3:** Дождитесь завершения
- Скрипт покажет прогресс
- После завершения вы увидите статистику
- Откройте дашборд в браузере (F5 для обновления)

## 🛠️ Автоматическое обновление (опционально)

### Создание .bat файла для быстрого запуска

**Файл:** `update_database.bat`

```batch
@echo off
chcp 65001 >nul
echo ============================================================
echo Обновление базы данных из Excel
echo ============================================================
echo.

cd /d "%~dp0"
python import_excel_to_pg.py

echo.
echo ============================================================
echo Нажмите любую клавишу для выхода...
pause >nul
```

**Использование:**
- Просто дважды кликните на `update_database.bat`
- Скрипт выполнится автоматически

## ❓ Частые вопросы

### Q: Нужно ли останавливать сервер при обновлении?
**A:** Нет, скрипт обновит данные, а backend автоматически подхватит изменения через 5 минут (кэш).

### Q: Что если скрипт выдал ошибку?
**A:** Проверьте:
1. Закрыт ли Excel файл
2. Правильно ли подключён PostgreSQL
3. Есть ли файл `database.xlsx` в папке проекта

### Q: Можно ли обновлять данные во время работы пользователей?
**A:** Да, но рекомендуется предупредить пользователей. Данные обновятся через 5 минут (после сброса кэша).

### Q: Как часто нужно обновлять данные?
**A:** По вашему сценарию — **1 раз в неделю**.

## 🔧 Настройка (для администратора)

### Переменные окружения (.env)

```bash
# Путь к Excel файлу
EXCEL_FILE_PATH=database.xlsx

# PostgreSQL
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_USER=postgres
POSTGRES_PASSWORD=ваш_пароль
POSTGRES_DATABASE=cgm_dashboard
```

### Требования

- Python 3.14+
- PostgreSQL 17+
- Установленные зависимости:
