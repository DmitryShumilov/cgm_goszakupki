## Сценарии использования

### Сценарий 1: Анализ по годам
- [ ] Выбрать фильтр "Год = 2025"
- [ ] Проверить, что общая сумма обновилась
- [ ] Проверить, что динамика по месяцам показывает 2025 год
- [ ] Проверить, что топ регионов показывает данные за 2025

### Сценарий 2: Анализ поставщика
- [ ] Выбрать фильтр "Поставщик = Медиалайн"
- [ ] Проверить, что общая сумма показывает долю Медиалайн
- [ ] Проверить, что топ заказчиков показывает работающих с Медиалайн
- [ ] Проверить, что тепловая карта показывает товары Медиалайн

### Сценарий 3: Анализ категории товара
- [ ] Выбрать фильтр "Что закупали = Freestyle Libre"
- [ ] Проверить, что общая сумма показывает объём категории
- [ ] Проверить, что топ регионов показывает где закупали Freestyle Libre
- [ ] Проверить, что динамика показывает закупки по месяцам

### Сценарий 4: Региональный анализ
- [ ] Выбрать фильтр "Регион = Москва"
- [ ] Проверить, что общая сумма показывает закупки в Москве
- [ ] Проверить, что топ заказчиков показывает заказчиков Москвы
- [ ] Проверить, что топ поставщиков показывает кто поставляет в Москву

### Сценарий 5: Комбинированный фильтр
- [ ] Выбрать "Год = 2024, 2025" + "Регион = Москва, СПб" + "Поставщик = Медиалайн"
- [ ] Проверить, что все данные отфильтровались корректно
- [ ] Проверить, что KPI показывает сумму по выбранным критериям
- [ ] Сбросить фильтры
- [ ] Проверить, что данные вернулись к исходным
