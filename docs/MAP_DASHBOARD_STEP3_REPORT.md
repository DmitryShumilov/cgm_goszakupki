## 📁 Созданные файлы

### 1. Стили (`src/styles/map.css`)

**Основные компоненты:**
- `.map-wrapper` — контейнер на всю высоту экрана с градиентным фоном
- `.region-path` — стили для регионов (hover, selected, no-data)
- `.map-legend` — легенда карты с градиентом
- `.region-info-panel` — панель детализации региона
- `.kpi-card` — карточки KPI в стиле основного дашборда

**Цветовая схема (из основного дашборда):**
- Фон: `linear-gradient(135deg, #0f0c29 0%, #302b63 50%, #24243e 100%)`
- Акцент: `#3388ff` (синий)
- Выделение: `#ff9800` (оранжевый)
- Текст: `#ffffff` с прозрачностью

---

### 2. Компонент карты (`src/components/Map/Map.tsx`)

**Функционал:**
- Загрузка GeoJSON из `/public/russia.geojson`
- Отображение регионов с цветовой кодировкой
- Hover-эффект с тултипом
- Клик для выбора региона
- Подсветка выбранного региона

**Логика окрашивания:**
```typescript
// Цвет зависит от суммы закупок
const intensity = data.sum / maxSum;
const opacity = 0.3 + (0.9 - 0.3) * intensity;
return `rgba(51, 136, 255, ${opacity})`;
```

---

### 3. Легенда карты (`src/components/Map/MapLegend.tsx`)

**Отображает:**
- Градиент от минимальной до максимальной суммы
- 4 уровня интенсивности (100%, 66%, 33%, 0%)
- Количество регионов без данных

**Форматирование:**
- `> 1 млрд` → "1.2 млрд ₽"
- `> 1 млн` → "500 млн ₽"
- `< 1 млн` → "250 тыс ₽"

---

### 4. Детализация региона (`src/components/RegionDetail/RegionDetail.tsx`)

**Панель показывает:**
- Название региона
- 4 KPI карточки:
  - Общая сумма
  - Количество контрактов
  - Средний контракт
  - Общий объём (шт)
- Заглушка для будущей детализации (топ поставщиков, категорий)

---

| ... | ... | ... |

---

### 6. Store (`src/stores/mapStore.ts`)

**Состояние:**
- `selectedRegion` — выбранный регион
- `regionData` — данные по регионам
- `isLoading` — статус загрузки
- `error` — ошибка

**Действия:**
- `setSelectedRegion(region)` — выбрать регион
- `clearSelection()` — сбросить выбор
- `loadRegionData()` — загрузить данные

---

### 7. Главный компонент (`src/App.tsx`)

**Структура:**
```tsx
<header>
  🗺️ CGM Dashboard — Карта закупок
  Интерактивная карта госзакупок по регионам России
</header>

<Map 
  onRegionSelect={setSelectedRegion}
  regionData={regionData}
  selectedRegion={selectedRegion}
/>

<MapLegend regionData={regionData} />

{selectedRegion && (
  <RegionDetail 
    region={selectedRegion}
    data={selectedRegionData}
    onClose={clearSelection}
  />
)}
```

## 🎨 Визуальные решения

### Цветовая схема (из основного дашборда)

```
Фон страницы:  #0f0c29 → #302b63 → #24243e (градиент)
Регионы:       #3388ff (синий с прозрачностью)
Выделение:     #ff9800 (оранжевый)
Текст:         #ffffff (белый)
KPI акцент:    #4fc3f7 (голубой)
```

### Анимации

- **Hover региона:** плавное изменение `fillOpacity` (0.3s)
- **Выбор региона:** изменение цвета на оранжевый
- **Панель региона:** появляется справа с тенью
- **Загрузка:** spinner с анимацией вращения

### Адаптивность

```css
@media (max-width: 768px) {
  .region-info-panel {
    position: fixed;
    bottom: 0;
    width: 100%;
    max-height: 60vh;
  }
}
```
