## 📋 Рекомендации

### Приоритет 1 (Критично — 6 часов)

#### 1.1 Исправить `logger` до объявления (5 мин)

**Файл:** `backend/main.py`

**Проблема:**
```python
@app.on_event("shutdown")
def shutdown_db_pool():
    logger.info("Database connection pool closed")  # ← logger ещё не определён
```

**Решение:** Переместить `logging.basicConfig()` выше `@app.on_event`.

#### 1.2 Создать backend тесты (2 часа)
