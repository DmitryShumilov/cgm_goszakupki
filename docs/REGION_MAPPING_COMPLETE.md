## 🔄 Как работает сопоставление

```
┌─────────────────────────────────────────────────────────────┐
│                     GeoJSON (simple_map.geojson)            │
│                     region.properties.name = "Москва"       │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│              normalizeRegionName("Москва")                  │
│              → поиск в regionMapping                        │
│              → "Москва" (без изменений)                     │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│              База данных (purchases.region)                 │
│              WHERE region = "Москва"                        │
│              → поиск данных                                 │
│              → { sum: 1.25 млрд, count: 234 }               │
└─────────────────────────────────────────────────────────────┘
```

## 🎯 Примеры работы

### Пример 1: Регион без изменений

```typescript
// GeoJSON
feature.properties.name = "Москва"

// Сопоставление
normalizeRegionName("Москва") → "Москва"

// Поиск в БД
regionData.find(r => r.region === "Москва") → найдено
```

### Пример 2: Регион с изменением

```typescript
// GeoJSON
feature.properties.name = "Ханты-Мансийский автономный округ"

// Сопоставление
normalizeRegionName("Ханты-Мансийский автономный округ") 
  → "Ханты-Мансийский автономный округ - Югра"

// Поиск в БД
regionData.find(r => r.region === "Ханты-Мансийский автономный округ - Югра") 
  → найдено
```

### Пример 3: Новый регион (2024)

```typescript
// GeoJSON
feature.properties.name = "Донецкая область"

// Сопоставление
normalizeRegionName("Донецкая область") → "Донецкая область"

// Поиск в БД
regionData.find(r => r.region === "Донецкая область") → найдено
```
