# 🗺️ Файл сопоставления регионов

**Назначение:** Приведение названий регионов из GeoJSON к виду в базе данных

---

## 📁 Файл

**Путь:** `frontend_map/src/utils/regionMapping.ts`

---

## 🎯 Проблема

Названия регионов в разных источниках могут отличаться:

| Источник | Пример названия |
|----------|-----------------|
| **GeoJSON** | `Ханты-Мансийский автономный округ` |
| **База данных** | `Ханты-Мансийский автономный округ - Югра` |
| **GeoJSON** | `г.Севастополь` |
| **База данных** | `Севастополь` |
| **GeoJSON** | `Республика Северная Осетия` |
| **База данных** | `Республика Северная Осетия - Алания` |

---

## 🔧 Решение

Файл `regionMapping.ts` содержит словарь для преобразования названий.

### Структура

```typescript
export const regionMapping: Record<string, string> = {
  // Ключ: название из GeoJSON
  // Значение: название для БД
  'Ханты-Мансийский автономный округ': 'Ханты-Мансийский автономный округ - Югра',
  'г.Севастополь': 'Севастополь',
  'Республика Северная Осетия': 'Республика Северная Осетия - Алания',
  // ...
};
```

---

## 📊 Экспортные функции

| Функция | Назначение |
|---------|------------|
| `normalizeRegionName(geojsonName)` | GeoJSON → БД |
| `toGeojsonName(dbName)` | БД → GeoJSON |
| `getGeojsonRegions()` | Список всех регионов GeoJSON |
| `getDbRegions()` | Список всех регионов БД |

---

## 🔄 Как используется

### В компоненте карты (`Map.tsx`)

```typescript
import { normalizeRegionName } from '../../utils/regionMapping';

// При клике на регион
const geojsonRegionName = feature.properties.name;  // из GeoJSON
const normalizedDbName = normalizeRegionName(geojsonRegionName);  // для БД

// Поиск данных
const data = regionData.find(r => r.region === normalizedDbName);
```

## 📋 Полный список сопоставлений

### Особые случаи

| GeoJSON | База данных | Причина |
|---------|-------------|---------|
| `г.Севастополь` | `Севастополь` | Убран префикс "г." |
| `Ханты-Мансийский автономный округ` | `Ханты-Мансийский автономный округ - Югра` | Добавлено "- Югра" |
| `Республика Северная Осетия` | `Республика Северная Осетия - Алания` | Добавлено "- Алания" |
| `Ямало-Ненецкий автономный округ` | `Ямало-Ненецкий автономный округ` | Без изменений |

### Типовые соответствия (без изменений)

Большинство регионов имеют одинаковые названия:

- `Москва` → `Москва`
- `Санкт-Петербург` → `Санкт-Петербург`
- `Республика Татарстан` → `Республика Татарстан`
- `Краснодарский край` → `Краснодарский край`
- `Свердловская область` → `Свердловская область`

## 📝 Добавление нового региона

Если в базе данных появился новый регион:

1. Откройте `frontend_map/src/utils/regionMapping.ts`
2. Добавьте запись:

```typescript
'Название из GeoJSON': 'Название в БД',
```

3. Сохраните файл
4. Перезапустите dev-сервер

## ⚠️ Важные замечания

1. **Порядок имеет значение:** Если регион не найден в mapping, используется исходное название
2. **Обратное сопоставление:** Генерируется автоматически из `regionMapping`
3. **Новые регионы:** При добавлении в БД обновите mapping
