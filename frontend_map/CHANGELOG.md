## [1.1.0] — Март 2026

### 🔗 Интеграция с backend
- ✅ Полная интеграция с FastAPI backend (порт 8000)
- ✅ Реальные данные из PostgreSQL через API
- ✅ Автоматическое обновление данных при изменении фильтров
- ✅ Исправлен CORS (добавлен localhost:5174)

### 🗺️ Mapping регионов
Исправлено 10+ несоответствий между GeoJSON и БД:

**Города:**
- `Санкт-Петербург` ↔ `Санкт - Петербург`

**Автономные округа:**
- `Ханты-Мансийский автономный округ — Югра` ↔ `Ханты - Мансийский АО`
- `Ямало-Ненецкий автономный округ` ↔ `Ямало - Ненецкий АО`
- `Чукотский автономный округ` ↔ `Чукотский АО`

**Республики:**
- `Удмуртская Республика` ↔ `Республика Удмуртия`
- `Чеченская Республика` ↔ `Чеченская республика`
- `Чувашская Республика` ↔ `Республика Чувашия`
- `Кабардино-Балкарская Республика` ↔ `Республика Кабардино - Балкария`
- `Карачаево-Черкесская Республика` ↔ `Республика Карачаево Черкессия`
- `Республика Северная Осетия — Алания` ↔ `Республика Северная Осетия Алания`

### 🎛️ Фильтры
- ✅ Кнопка "Выбрать всё" перемещена в dropdown (вместо отдельной кнопки)
- ✅ Чекбоксы для каждого значения в dropdown
- ✅ Автоматическое переключение "Выбрать всё" ↔ "Сбросить всё"

### 🐛 Исправления
- ✅ KPI обновляются автоматически при изменении фильтров
- ✅ Все 85 регионов отображают корректные данные
- ✅ Санкт-Петербург показывает ~1.46 млрд ₽
