# 🗺️ CGM Dashboard — Карта закупок

Интерактивная карта госзакупок по регионам России с аналитикой и фильтрами.

**Статус:** ✅ Production Ready  
**Оценка UI/UX:** 90/100  
**Доступность:** WCAG 2.1 AA (88/100)  
**Интеграция с backend:** ✅ Полная  
**Последнее обновление:** Март 2026

---

## ✨ Возможности

- 🌍 **Интерактивная карта** — все регионы РФ (85 регионов) с цветовой кодировкой по сумме закупок
- 🎛️ **Умные фильтры** — по годам, продуктам, поставщикам с видимыми чипами и кнопкой "Выбрать всё"
- 📊 **Детализация** — панель региона с KPI (4 метрики), tooltip и экспортом в CSV
- 🎨 **Тёмная тема** — современный дизайн в стиле CGM Dashboard (CSS переменные, 8px сетка)
- ♿ **Доступность** — WCAG 2.1 AA, keyboard navigation (Tab, Enter, Escape), skip link
- 📱 **Адаптивность** — корректное отображение на всех разрешениях (mobile/tablet/desktop)
- 🔗 **Интеграция с backend** — реальные данные из PostgreSQL через FastAPI API

## 🔗 Интеграция с backend

**Статус:** ✅ Полная интеграция (Март 2026)

### API Endpoints

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/map/regions` | GET | Данные для карты всех регионов |
| `/api/map/regions?years=2024,2025` | GET | С фильтрами по годам |
| `/api/filters/years` | GET | Список доступных лет |
| `/api/filters/regions` | GET | Список регионов |
| `/api/filters/suppliers` | GET | Список поставщиков |
| `/api/filters/products` | GET | Список продуктов |

### Пример запроса

```bash
curl http://localhost:8000/api/map/regions
```

**Ответ:**
```json
[
  {
    "region": "Санкт - Петербург",
    "sum": 1463108100.0,
    "count": 18,
    "quantity": 388446,
    "avg_price": 81283777.43
  },
  ...
]
```

### Настройка подключения

**Файл:** `.env`
```env
VITE_API_URL=http://localhost:8000
```

## 🗺️ Как работает карта

1. **Загрузка GeoJSON** — регионы России из `/russia_regions.geojson`
2. **Нормализация названий** — сопоставление GeoJSON → БД через `regionMapping.ts`
3. **Цветовая кодировка** — градиент интенсивности по сумме закупок
4. **Интерактивность** — ховер с тултипом, клик открывает панель региона

## 🎛️ Фильтры

| Фильтр | Шрина | Описание |
|--------|-------|----------|
| **Год** | 180px | Выбор одного или нескольких лет с выделением счётчика |
| **Продукты** | 220px | Выбор медицинских продуктов с обрезкой названий |
| **Поставщик** | 220px | Выбор поставщиков с обрезкой названий |

### Особенности фильтров

- ✅ **Видимые чипы** — отображение выбранных значений с выделением счётчика (+N)
- ✅ **Выделение счётчика** — рамка и фон на 1-2 тона светлее
- ✅ **Обрезка названий** — 30 символов + "..." для длинных названий
- ✅ **Сохранение в localStorage** — фильтры сохраняются между сессиями
- ✅ **Кнопка сброса** — быстрая очистка всех фильтров
- ✅ **Активные фильтры** — панель с чипами всех выбранных фильтров

## 📊 Панель региона

При клике на регион открывается панель с:

### KPI карточки
- 💰 **Общая сумма** — с tooltip и выделением
- 📋 **Контрактов** — количество заключённых контрактов
- 📈 **Средний контракт** — расчёт: Общая сумма / Количество
- 📦 **Объём (шт)** — общий объём закупленных товаров

### Функции
- 🔔 **Tooltip** — подсказки для каждой метрики (ℹ️)
- 💾 **Экспорт CSV** — выгрузка данных региона в CSV формат
- ⌨️ **Keyboard support** — закрытие по Escape
- 🎬 **Анимация** — плавное появление (slideInRight)

## ♿ Доступность (WCAG 2.1 AA)

### Keyboard Navigation
| Клавиша | Действие |
|---------|----------|
| `Tab` | Навигация между элементами |
| `Enter/Space` | Активация кнопок (зум, фильтры) |
| `Escape` | Закрытие панели региона |

### ARIA атрибуты
- ✅ `role="banner"` для хедера
- ✅ `role="dialog"` для панели региона
- ✅ `role="complementary"` для легенды
- ✅ `aria-label` для всех интерактивных элементов
- ✅ `aria-live="polite"` для счётчика регионов

### Skip Link
- ✅ Появляется при нажатии `Tab` в начале страницы
- ✅ Переход к основному контенту (карте)

### Reduced Motion
- ✅ Поддержка `prefers-reduced-motion`
- ✅ Отключение анимаций для пользователей с предпочтением
